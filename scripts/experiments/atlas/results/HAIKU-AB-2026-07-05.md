# Haiku 4.5 vs Fable 5 — atlas classification A/B (2026-07-05)

**Verdict: Haiku 4.5 holds. Classification quality is within noise of
Fable 5, so the corpus-scale cost model (a small model at cents per 1000
papers) is validated.** This was the last cheap go/no-go input before a
corpus pilot.

Same 54 papers, same prompts, same isolation, same criteria. Only the
model differs (Fable 5 vs Haiku 4.5, both via subagents).

| metric | Fable 5 | Haiku 4.5 |
|---|---|---|
| C1 classification recall | 0.93 (28/30) | **0.90 (27/30)** |
| C2 join, strict | 9/15 | 9/15 |
| C2 join, equivalence-class | 14/15 | **13/15** |
| distractor assignments (raw count) | 15 | **10** (more conservative) |
| planted papers classified differently | — | 4/30 |

## The 4 planted papers where the models disagree

- **2004.01025** (mirror descent): Fable → mirror_descent, Haiku →
  gradient_descent. A near-miss (both are gradient-based updates); costs
  Haiku the replicator↔mirror-descent join. Defensible, not nonsense.
- **1612.08949** (logistic adoption): Fable → binary_spin_energy, Haiku →
  replicator_dynamics. Both WRONG (expected logistic_growth); this paper
  is a genuine hard case (opinion dynamics with contrarian agents) that
  both model tiers misread differently.
- **1506.01137** (logistic growth): Fable → [logistic_growth,
  master_equation], Haiku → [logistic_growth]. Haiku dropped a defensible
  secondary tag; the primary (correct) tag is identical.
- **2208.02457** (Lotka-Volterra): Fable → [lotka_volterra,
  hamiltons_equations], Haiku → [lotka_volterra]. Haiku dropped a
  spurious secondary tag Fable added — arguably Haiku is *cleaner* here.

## Reading

Two of Haiku's four differences are it being more conservative (dropping
secondary tags), which is neutral-to-good. One is a true near-miss
(mirror descent). One is a hard case both tiers miss. On the primary
classification decision, Haiku and Fable agree on 29/30 planted papers.
On distractors, Haiku assigned fewer templates (10 vs 15) — i.e. slightly
higher precision / more willing to say "no template fits."

Net: no meaningful quality loss at ~1/20th the token cost. A full-corpus
atlas run is economically trivial. **Use Haiku (or a small model) for the
corpus pilot; reserve a frontier model only for the moderated / published
tier if desired.**

## Caveat

Still a 54-paper gold standard authored by the same model family. The
corpus pilot (unlabeled real papers) is the next test of whether
precision holds when the paper is NOT one a template was implicitly
written for.
