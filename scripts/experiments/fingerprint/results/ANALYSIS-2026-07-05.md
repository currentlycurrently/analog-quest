# Analysis — first fingerprint recall run (2026-07-05)

**Verdict: pre-registered criterion NOT MET (recall@25 = 5/12 = 0.42 vs
required 0.50), but the substrate shows strong, diagnosable signal and a
clear v2 direction. Neither "ship it" nor "abandon it" — iterate once.**

Fingerprinting model: Claude (Fable 5) via Claude Code subagents,
2026-07-05, prompts generated by `fingerprint.py --emit-prompts` from
`prompts.py` (so the API route reproduces the same inputs). 47 papers
(24 planted = 12 known-isomorphism pairs, 23 distractors), 966
cross-archive candidate pairs.

| metric | value | random baseline |

| recall@10 | 5/12 | ~0.1 pairs expected |
| recall@25 | 5/12 | ~0.3 pairs expected |

Five known isomorphisms ranked in the **top 7 of 966**: SIR↔rumor (#1),
Burgers↔traffic (#2), Black–Scholes↔heat (#4), Kuramoto↔power-grid (#6),
optimal-transport↔matching (#7). For comparison, the exact-hash substrate
this experiment challenges would have ranked all 12 pairs nowhere — none
of these papers share a byte-identical normalized equation.

(median: 57 / 27.5 / 33; recall@25 is 5/12 for all three variants)

**Finding 1 — the categorical agreement terms hurt.** Exact-match scoring
on linearity/stochasticity/spatial-structure penalizes precisely the pairs
where two fields dress the same skeleton differently (deterministic ODE vs
stochastic variant, continuum vs agent-based). Features-only rescues
lotka_volterra_econ (122→9) and halves the ising/logistic ranks.

**Finding 2 — one fingerprint per paper forces a framing choice that
splits equivalent pairs.** The four persistent misses all have the same
shape:

- `ising_hopfield`: Ising side fingerprinted as `statistical_model`
  (equilibrium framing), Hopfield side as `network_dynamics` (retrieval
  framing). Same energy function, two faces.
- `logistic_adoption`: biology side is a *stochastic* logistic paper
  (`stochastic_process`), adoption side an *agent-based discrete* model
  (`discrete_update`). The shared logistic ODE skeleton is buried under
  each paper's dressing.
- `replicator_mirror_descent`: continuous-time stochastic replicator vs
  discrete-time optimization update. Same story.
- `oscillator_rlc`: ground-truth curation error, not substrate failure —
  the chosen RLC paper ("Monotone RLC Circuits") is really an
  operator-theory/optimization paper, so its core model genuinely isn't
  the damped-driven oscillator. Replace it in v2.

**Finding 3 — precision looks acceptable at this scale.** The non-planted
pairs in the top 10 are structurally reasonable near-analogies (e.g.
mirror descent ↔ a math.OC discrete-update paper; SIR ↔ predator-prey,
which are both mass-action compartmental ODEs — arguably a *true*
unplanned analogy, exactly what the real pipeline would want to surface).
The top of the list is not garbage. No formal precision measurement was
done at this distractor density; do one in v2.

## v2 changes implied (in order)

1. **Multi-view fingerprints.** Add a `deterministic_skeleton` /
   `backbone` field: "if you strip noise, discretization, and equilibrium
   framing, what is the underlying dynamical/variational skeleton?"
   Match across views, score on the best view-pair. This directly attacks
   all three real misses.
2. **Drop or soften the categorical agreement terms**; make
   feature-Jaccard (or feature-overlap weighted by rarity) carry the
   score, with object-type as a soft prior only.
3. **Fix the oscillator_rlc ground-truth entry** (pick a paper whose core
   model is actually RLC dynamics), and add 2–3 held-out analogies chosen
   by someone who has not seen the feature vocabulary (bias control noted
   in README).
4. **Then re-run.** Same pre-registered criterion, same distractor set.
   If v2 clears recall@25 ≥ 0.5, graduate to a 500-paper corpus run with
   precision moderation; if it doesn't, the honest conclusion is that
   fingerprint matching needs a pairwise LLM verification stage even at
   the *candidate generation* level, which changes the cost model — decide
   then.

## Honest caveats

- One planted paper (2004.09314) extracted 0 equations (LaTeX quirk);
  its fingerprint is abstract-only. It still hit rank #1 (sir_rumor),
  which says something good about abstract-level structure extraction but
  means that pair didn't test the equation pathway.
- The fingerprinting model (Fable 5) is a frontier model; the production
  cost model assumes a small model (Haiku-class). v2 should A/B at least
  a few papers to see if fingerprint quality survives the downgrade.
- 23 distractors is a small decoy set. Ranks are informative relative to
  random baseline, but corpus-scale false-positive density is unmeasured.
- Feature vocabulary and ground truth were authored by the same agent
  (bias documented in README). The five top-7 hits are robust to this
  concern (they'd survive any reasonable vocabulary); the exact recall
  numbers are not.
